# E-Commerce Platform

A dynamic and secure E-Commerce web application developed using C# and the ASP.NET framework. The platform provides a complete online shopping experience, featuring user management, product catalogs, and transactional workflows.

## 🚀 Key Features
* **User Authentication:** Secure registration, login, and session tracking for customers.
* **Product Catalog & Management:** Organized display of products categorized by types, with search and filtering capabilities.
* **Shopping Cart System:** Fully functional cart module allowing users to add, update, or remove items before proceeding to checkout.
* **Responsive UI:** Clean, intuitive layout crafted for seamless navigation across both desktop and mobile devices.

## 🛠️ Tech Stack
* **Backend:** C# (.NET Framework)
* **Frontend:** HTML5, CSS3, JavaScript, and Bootstrap
* **Database:** SQL Server / MySQL (Relational Database)
* **IDE:** Visual Studio
